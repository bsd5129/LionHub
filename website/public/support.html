Support

The main purpose behind LionHub is to make life easier for students. With that in mind, we have provided an online tutorial with screenshots to assist students as they learn to use all of the available features of LionHub.

This is the Home Screen for LionHub. Everything can be accessed from this location.

Selecting Services from the Home Screen brings you to the Services Menu. This contains Student Services broken down into several categories.

Pressing the Online option from the Services Menu takes you to a list of Online Services available to the student.

You have the option of pressing Back to navigate back to the Services Menu.

Pressing on one of the Services will bring up a screen containing a description of the Service, a phone number, and a web link for further exploration.

There are going to be certain Services that you plan to use often. Rather than navigate through all the Services each time, you can add a Service to your Favorites.

Now, you can go straight to these Favorite Services by selecting Favorites from the Home Screen.

Another Feature of LionHub is the Rooms Directory. You can access it from the Home Screen by selecting Rooms.

You can enter a room number and select the building from a drop down menu and search for any room on campus.

Pressing Search will bring up all important information on the room.

Or it can bring up a message saying this room has not been found.

A Scavenger Hunt can be reached from the Home Screen as well. Some of the clues require text responses and some require that you be in a certain location.

You will know if a text response is required if there is a text box on the screen. The correct solution must be typed here to move on to the next clue.

Otherwise a message explaining you must go to the clue's location will be displayed. In all cases, you can press the Extra Hint Button for additional clues.

If the wrong answer is typed or if you are at the wrong location, the appropriate message will appear when you press Check Solution.

Another Feature of LionHub is the Calendar of Events. This can be accesed from the Home Screen.

When you press Events on the Home Screen, you are brought to the Events screen. As the meassage states, ther may be a brief delay while Events are loaded.

The Calendar defaults to showing the current month's Events.

Pressing on the month brings up a Dropdown Menu allowing you to select any month containing Events

The Where Am I Feature locates the building nearest to your location and displays relevant information.

As the building is being located, a screen informs you of it's search.

An alert indicating you are too far away is displayed when you are too far from the Penn State Harrisburg campus.

The last option on the Home Screen is the Settings Menu. It is accesed by pressing Settings.

The Dark Theme allows you to change the look and feel of LionHub.

Pressing Credits brings up information about a few hard working college students.

LionHub periodically checks for Updates. Upon opening LionHub, a message will be displayed if a new version is available for download.
